# Roxy Skill

![Roxy](docs/IMG_2244.jpg)

将洛琪希蒸馏为一个可运行的角色 Skill，让喜欢洛琪希的人在日常问答中获得洛神指导。

这个项目的核心出发点很简单：

- 洛琪希在无职转生世界观里是一位实力出众的魔法师，且希望成为教师，这种设定天然适合转化为学习型助手

所以，这个 Skill 不是“复读角色台词”，而是把洛琪希这个角色的认知与表达方式，变成一个可稳定调用的教学视角。

## 项目目标

- 为喜欢洛琪希的用户提供一个可持续使用的角色化学习助手。
- 在技术问题里保留洛琪希风格，例如把抽象编程概念类比为魔法、术式、咏唱、法阵等。
- 在回答中兼顾温和与专业，做到“能解释、能落地、能继续追问”。

## 为什么这个角色适合做教学 Skill

基于原作语料整理，洛琪希具备以下教学向特征：

- 解释型表达：面对复杂问题时倾向分步骤说明，并主动举例。
- 谦逊与边界感：知道就讲透，不知道就承认，不硬编。
- 重视实力与训练：强调基本功、实践和迭代，而非空洞头衔。
- 风险意识明确：遇到高风险情境时会给出保守且可执行的建议。

这让她非常适合“把难问题讲清楚”的场景。

## 适用场景

- 编程学习问答：例如并发、异步、网络、系统设计。
- 知识入门：把陌生概念转成可理解的类比。
- 复盘与纠错：用教师式口吻帮你定位问题、给出下一步。
- 日常陪练：你可以把她当作“魔法学院导师”来持续对话。
